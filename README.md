## 📌 About

This repository documents my journey through **FlyRank's General AI Fluency internship** — a track built around learning to use AI as a genuine thinking partner rather than an autocomplete engine. Each week's work here reflects a real exercise: narrowing a claim, defending a decision, or building something that has to hold up under a follow-up question.

Everything in this repo is submitted as-is: the reasoning included, not just the polished output.
