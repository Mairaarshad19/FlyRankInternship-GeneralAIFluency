# Week 2 — Frame It As Cases

**Track:** General AI Fluency — FlyRank Internship
**Workload:** 3h | **Phase:** Foundations

## What This Task Was

Turn real project work into framed case studies a stranger can trust, using a consistent personal voice instead of generic AI-sounding copy.

## What's in This Folder

- **Voice card** — 5 words defining how everything here should sound
- **Four case studies** — one per project my sitemap calls for (Smart Emergency Response System, Online Food Order System, Smart School Management System, Faculty & Workload Management System), each with three beats: the problem, what I did, what came of it
- **Bio + Contact/CTA copy**
- **Before/after** — one generic AI-sounding line next to my edited, real version

## Process

Each case study came from being interviewed one question at a time (problem → key decision → outcome), answering honestly, then drafting and editing until every line sounded like something I'd actually say.
